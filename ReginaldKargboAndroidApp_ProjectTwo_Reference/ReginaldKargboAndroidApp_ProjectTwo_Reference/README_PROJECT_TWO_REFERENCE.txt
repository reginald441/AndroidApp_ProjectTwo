Project Two Reference/Learning Version

This Android Studio project demonstrates a Project Two-style inventory app structure:
- Login screen with username and password
- Create account screen
- SQLite database for users and inventory items
- Inventory data grid using a table layout
- Add item functionality
- Delete item functionality
- SMS notification permission screen
- Runtime SMS permission request

Important: This file was prepared as a reference/study version. Review the code, test it in Android Studio, and make changes in your own words/style before using it for coursework.
